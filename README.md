# Dotfiles

This repository contains dotfiles for my machine(s). This repository is managed with [chezmoi](https://github.com/twpayne/chezmoi).

## Migration note

- Migration strategy: I moved the previous configuration files into a .old directory at the root of this repo. I'll migrate functionality into the chezmoi source tree piece by piece, leaning on git to preserve per-file history as files are split or reorganized.
- The last pre-chezmoi version is commit [2ff6f80cdabf0a7da6fa5d36aef5de628c88481a](https://github.com/kevinjalbert/dotfiles/tree/2ff6f80cdabf0a7da6fa5d36aef5de628c88481a).
